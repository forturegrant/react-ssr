# create-react-practice

## ✨ 介绍
一个能够快速生成react/vue项目的脚手架

## 📦 安装

```bash
npm i create-react-practice -g   // 记得全局安装
```

## 🔨 使用

```bash
rp-cli create [project-name]
```
